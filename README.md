# Real Estate & Construction Website Showcase

A modern, responsive, and high-performance website designed for a Real Estate and Construction business. This project features a premium aesthetic with dark mode elements, glassmorphism effects, and smooth reveal animations.

## 🚀 Features

-   **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
-   **Modern UI/UX**: Premium aesthetic using a balanced color palette of Dark Navy, Gold, and Off-White.
-   **Smooth Animations**: Scroll-reveal effects for a dynamic user experience.
-   **Interactive Elements**:
    -   3D Flipping Service Cards.
    -   Responsive Navigation with active state highlighting.
    -   Mobile-friendly toggle menu.
    -   Scroll-to-top functionality.
    -   Floating WhatsApp and Call buttons for quick engagement.
    -   Functional FAQ accordion.
-   **Contact Integration**: Professional contact form and integrated Google Maps.
-   **SEO & Performance Optimized**:
    -   Configured `robots.txt` and `sitemap.xml`.
    -   Meta theme color integration.
    -   Image lazy loading and async decoding.
    -   Preconnect hints for faster asset delivery.

## 🛠️ Tech Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
-   **Icons**: Font Awesome 6.4.0
-   **Typography**: Google Fonts (Inter/Outfit suggested in CSS)
-   **Version Control**: Git

## 📄 Pages Included

The website consists of **5 main pages**:

1.  **Home (`index.html`)**: Overview of the business, featured services, and recent achievements.
2.  **About Us (`about.html`)**: Company journey, mission, vision, and core strengths.
3.  **Services (`services.html`)**: Detailed breakdown of residential, commercial, industrial, and construction services.
4.  **FAQ (`faq.html`)**: Answers to common queries regarding property dealings and construction.
5.  **Contact Us (`contact.html`)**: Multiple ways to reach out, including a form and office location details.

## ⚙️ Setup & Usage

To run this project locally:

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Open any `.html` file in your preferred web browser.
3. No build step or dependencies required (Pure HTML/CSS/JS).
